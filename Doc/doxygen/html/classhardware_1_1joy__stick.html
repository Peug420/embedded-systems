const uint16_t centerPoint_ {2047}
 The center of the ADC resolution. The raspberry pi pico has a a 10 bit by default and a 12 bit optionally. The 12 bit is used so the resolution ist 2^12 = 4096 of which 2047 is the half. This is used for calculating the dection radius and detection band.
 
const uint16_t centerDetectRadius_ {1000}
 The radius in which no input is detected. The radius is relative to the maximal resoultion 4096 of each axis. So a directional input will be detected only if in any x or y direction the value from the ADC is greater than the centerPoint_ + centerDetectRadius_ or less than centerPoint_ - centerDetectRadius_.
